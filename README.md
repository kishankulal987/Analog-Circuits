# Analog Circuits

A collection of analog circuit design experiments, simulations, and mini-projects implemented using the Sky130 PDK, Xschem, and Ngspice.

## Tools Used

* Xschem
* Ngspice
* Sky130 PDK
* Magic VLSI

## Projects

* MOSFET as a Switch
* MOSFET Output Characteristics
* MOSFET Transfer Characteristics
* MOSFET as a Current Source
* Current Mirror
* Differential Pair

## Objective

To build a strong foundation in Analog IC Design and understand the building blocks required for implementing a Current Steering DAC.
